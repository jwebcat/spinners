Spinners

Simple CSS3 rotation animations.

Download Spinners

GitHub Project       Getting Started

How It Works

Simply add the .spinner class to any object to apply a CSS3 rotation animation.

<img class="spinner" src="spin.png">

Browser Compatability

Spinners works on all modern browsers, and IE 10 and above.

Built by Chris Ferdinandi. Licensed under WTFPL.

Spinners is an add-on for Kraken, a lightweight, mobile-first boilerplate for front-end web developers. It works great as a standalone product, too.
